# -E-COMMERCE-APPLICATION-
🚀 Features

🛍️ Add to Cart / Remove from Cart

Backend: ASP.NET Core 8, C#

Frontend: Razor Pages   / MVC (Choose based on your implementation)

Database: SQL Server
